# 🤖 AI Chatbot (React + OpenAI)

An AI-powered chatbot built using React and OpenAI APIs.  
This project demonstrates how to integrate LLM capabilities into a modern frontend application with a clean and responsive UI.

---

## 🚀 Features

- 💬 Real-time AI chat interaction
- ⚡ Fast and responsive UI using React
- 🎨 Styled with Tailwind CSS
- 🔌 OpenAI API integration
- 📜 Conversation-based messaging flow
- ♻️ Reusable and scalable component structure

---

## 🛠️ Tech Stack

- **Frontend:** React (JavaScript / TypeScript)
- **Styling:** Tailwind CSS
- **API:** OpenAI API
- **State Management:** React Hooks
- **Build Tool:** (Vite / CRA — update based on your setup)
